# WeatherForecastAssignment

How to Run the Project
1. Clone the Repository:git clone <your-github-repository-link>
2. Install Required Libraries: pip install pandas numpy matplotlib seaborn scikit-learn scipy
3. Open the Notebook: jupyter notebook
Open: notebooks/WeatherForecast.ipynb
4. Run All Cells

Execute all notebook cells sequentially to:
Clean the dataset,
Perform EDA,
Detect anomalies,
Train forecasting models,
Evaluate model performance,
Generate visualizations.
